# 📦 HashSeal

> Simple file hash generator & verifier  
> 跨平台文件哈希生成与校验工具

---

## ✨ Features

- 🔐 Generate SHA256 hash files
- 🔍 Verify file integrity
- 🖥 Cross-platform:
  - macOS
  - Linux
  - Windows (basic support)

---

## 🚀 Usage

### macOS / Linux

```bash
./HashSeal.sh file1 file2
```

or drag files into terminal

⸻

Windows

Drag files onto:
```
HashSeal.bat
```

📂 Output
- Generates .sha256 files
- Format:
  
```
HASH  filename
```

⚠️ Notes
- Windows version may have limitations with complex file paths
- PowerShell version is under improvement
